# React Soil Sensor Dashboard

A React + TypeScript dashboard for visualizing soil sensor data from an embedded system.

## Features
- Real-time visualization of soil metrics (moisture, pH, EC, temperature)
- Interactive charts using Recharts
- Clean dashboard UI for monitoring system behavior

## Tech Stack
- React + TypeScript
- Vite
- Recharts
- Embedded system (ESP32 + RS-485 sensor)

## Overview
This project connects embedded system data to a modern web interface, allowing real-time monitoring and analysis of environmental conditions.
